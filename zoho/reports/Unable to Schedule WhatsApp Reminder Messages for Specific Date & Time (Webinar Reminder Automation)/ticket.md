# Ticket Report

## Ticket ID
98595000033756007

## Subject
Unable to Schedule WhatsApp Reminder Messages for Specific Date & Time (Webinar Reminder Automation)

## Description
Hi Gallabox Support Team,
We are using Gallabox for our weekly Diabetes Freedom Webinar funnel (brand: Diabetes Free Forever by Dr. Bhagyesh Kulkarni).
We have recently upgraded our account to access the Sequence feature, expecting it to help us schedule WhatsApp reminder messages for our webinar attendees.
However, we’ve noticed that the Sequence workflow only allows time-based triggers relative to enrollment (e.g., “Send after X hours/days from first message”).

Our requirement is to send reminder messages at fixed calendar times — for example:

3 days before the webinar (Thursday 10 AM)

1 day before the webinar (Saturday 10 AM)

On the day of the webinar (Sunday 9 AM)

Since new leads keep coming into our system continuously, using “X days after enrollment” doesn’t work for us — because all users receive messages at different times rather than before the fixed webinar date.
We also tried using Broadcast, but it only allows 250 contacts per day, which is not sufficient (we usually have 1000+ leads weekly).
👉 We need your guidance or a possible feature solution to:

Schedule WhatsApp reminder messages for a fixed date and time (not relative to enrollment).

Send these messages (including media/carousels) to our entire active lead list without the 250-message broadcast cap.

Please let us know if there’s a way to:

Create time-based automation for a fixed upcoming event, or

Temporarily lift the broadcast limit, or

Provide an alternative setup using API / Journey feature to achieve this.

Attached is a screenshot of our current setup for reference.
